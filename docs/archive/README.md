# Planning History Archive

这个目录用于存放已归档的历史规划文档。

归档原则：

- 文档内容仍有参考价值
- 但已经不是当前仓库的主执行口径
- 与现行文档存在较高重复，继续保留在主目录会增加理解成本

当前归档到这里的文档，主要属于“增强版平台方案”的历史演进稿。

## 当前文档分层

当前仓库中的文档建议按下面三层理解：

- `docs/`
  - 当前运行文档、协议文档、正式项目计划
  - 适合日常开发、使用、维护时直接查阅

- `docs/validation/`
  - 验收/收口清单与工程证据
  - 适合回归、发布、阶段状态确认时查阅

- `docs/archive/`
  - 历史规划、旧版路线图、参考设计稿
  - 适合追溯演进过程，不作为当前执行主口径

后续评审、汇报与执行时，建议优先参考：

- `docs/unified_delivery_execution_plan.md`
- `docs/runbook.md`
- `docs/validation/stage3_stage4_closure_checklist.md`
- `docs/validation/stage5_stage6_closure_checklist.md`
- `docs/validation/stage5_stage6_readiness_checklist.md`
- `docs/validation/stage7_groundwork_readiness_checklist.md`
- `docs/validation/stage7_groundwork_closure_checklist.md`

如果需要追溯早期方案的演进过程，再回看本目录中的归档文档。
